## Installation

```bash
git clone https://github.com/HanmaDevin/cosmicdev.git "$HOME/cosmicdev/"
```

```bash
cd $HOME/cosmicdev/
```

```bash
./install.sh
```

## Upgrading packages

fzf, neovim, fastfetch and lazygit can be updated via
the scripts in the `scripts` directory

### Nvidia

The install script scans for nvidia hardware and installs the driver for you.

You should be good to go afterwards. :shipit:
